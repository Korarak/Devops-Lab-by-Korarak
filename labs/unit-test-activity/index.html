Jest In Depth

เจาะลึก Jest: AAA Pattern, จัดกลุ่ม Test ด้วย describe, Matchers และ Edge Cases

  • AAA Pattern (Arrange-Act-Assert)
  • describe() + it() — จัดกลุ่ม Test
  • Matchers สำคัญ 10+ รายการ
  • Test Edge Cases + toThrow()
  • beforeEach / afterEach — จัดการ State
