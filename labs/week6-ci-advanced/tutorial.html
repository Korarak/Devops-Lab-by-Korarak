  • TaskFlow API + Frontend (Vue 3)

  • 4-job pipeline: +Sonar job หลัง Test

  • Image tagging strategy: sha + latest
  • GitHub Actions cache type=gha
  • Conditional build: push เฉพาะ main/develop

ทักษะใหม่ที่จะได้เรียนรู้ใน Week 6

🔬
SonarQube Code Analysis
ตรวจ code smells, bugs, vulnerabilities และ coverage โดยอัตโนมัติทุก push
🚦
Quality Gate
กำหนด threshold — ถ้า coverage < 60% หรือมี Critical bug → pipeline fail
🐳
Docker BuildKit + Cache
สร้าง image เร็วขึ้นด้วย layer cache จาก GitHub Actions (type=gha)
🏷️
Image Tagging Strategy
tag ด้วย sha-abc1234 (traceable) + latest สำหรับ deploy ล่าสุด
⚙️
Conditional Jobs
Build & Push ทำงานเฉพาะ branch main/develop เพื่อลด cost
🔐
Secrets Management
เก็บ SONAR_TOKEN, REGISTRY_URL, credentials ใน GitHub Secrets
