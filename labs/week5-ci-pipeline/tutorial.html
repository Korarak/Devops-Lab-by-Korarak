🆕 สิ่งที่เรียนใน Week 5 — CI Pipeline ที่แท้จริง

🧪
Unit Tests (Jest + Supertest)
test /health, /api/tasks CRUD, validation
🎭
Mock Database
jest.mock('../db') — test โดยไม่ต้องมี DB จริง
📊
Code Coverage ≥ 60%
jest --coverage + threshold ใน jest.config.js
🔍
ESLint Config สมบูรณ์
.eslintrc.json + npm run lint ผ่านใน CI
⚙️
CI Workflow 3 Jobs
lint → test (+ coverage) → docker build
📦
Upload Coverage Artifact
actions/upload-artifact@v4 เก็บ HTML report
🛡️
CI Gate
PR ถูก block ถ้า test ยัง fail
🐛
Debug Challenge
ฝึกอ่าน CI log + แก้ broken test
