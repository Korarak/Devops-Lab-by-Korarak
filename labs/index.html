สัปดาห์ที่ 4
GitFlow & CI/CD GitHub Actions
ตั้งค่า GitFlow branching strategy, Branch Protection, Pull Request workflow และสร้าง CI/CD pipeline อัตโนมัติด้วย GitHub Actions
GitFlow GitHub Actions CI/CD Branch Protection Docker Push

  • เปิด Pull Request และทำ Code Review แบบ professional
  • เขียน CI workflow ที่ build และ test อัตโนมัติ
  • เขียน CD workflow ที่ push Docker image สู่ Registry เมื่อ release

Continuous Integration: First Pipeline
